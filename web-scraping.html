7.2 Scrape a specific item from a webpage

Right-click and inspect a page to investigate how it’s structured. Right-click an item and copy x-path If the path has quotes in it, use alternative form of quotes (single instead of double or vis a versa)

/html/body/div/main/article/section/div/div/p[4]

all_text <- urloi |> 
  read_html() |> 
  html_element(xpath = "/html/body/div[2]/div/div[3]/main/div[3]/div[3]/div[1]/figure[1]/figcaption") |> 
  html_text()

7.3 Ethics of web scraping

In the United States, web scraping is legal. However, there are many illegal things you may do with the data you scraped, including copyright infringement. Poor scraping design may also crash web servers through denial of service. add /robots.txt to a web address to see what you are not supposed to crawl.

Web scrapers can add pauses into their code to detect scraping behavior. Rtweet was throttled and re-queried every 12 to 15 minutes.

brady_data_clean_types <- brady_data_clean |> 
  slice(-1) |> 
  filter(`Year Year` != "Career") |> 
  mutate(

    year_numeric = as.numeric(`Year Year`)

brady_data_clean_types |> 
  ggplot() +
  geom_col(aes(x = year_numeric, y = passing_yards_numeric))
